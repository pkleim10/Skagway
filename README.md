# VideoMaster

macOS app to browse, tag, rate, and organize video files from folders on your disk.

**Documentation note**: A full user guide is intentionally minimal for now and will be produced closer to a production release. For current usage, explore the in-app UI, toolbar, menus, and Settings. Direction and planned work live in `ROADMAP.md`. Per-deployment history (version + build) is in `CHANGELOG.md`.
